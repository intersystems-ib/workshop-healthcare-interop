# install

This folder contains workshop setup assets and reusable configuration files.

- `simple-fhir-data/` contains sample FHIR resources that can be loaded into the FHIR repository
- `HL7/` contains HL7 demo schemas, rules, and related support files
- `roles.xml` and `webapps.xml` contain IRIS configuration artifacts used during setup
- `LabWorkflowResultSoapService.wsdl` contains the SOAP contract used by the Lab Workflow HIS result service example
- `fhirtransf.json` is a sample transformation spec for FHIR SQL Builder
- `mysql-connector-j-8.4.0.jar` supports the external MySQL catalog integration

The main manual steps that use this folder are documented in the top-level [README.md](../README.md).
