# ns-fhirrepo

This folder contains the FHIR repository and FHIR interoperability demo code.

- `src/fhirdemo` contains the production, business services, business processes, and helper classes used when FHIR requests are routed through Interoperability

Relevant workshop areas:

- Scenario 2 in the top-level [README.md](../README.md)
- sample requests in [http/fhir-repository.http](../http/fhir-repository.http)
- sample FHIR resources in [install/simple-fhir-data/](../install/simple-fhir-data/)
